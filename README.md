## 🚀 Project Overview

**LinkUp HK** is a conceptual AI-powered social wellness platform designed to help people in Hong Kong connect, build meaningful relationships, and improve their mental & social well-being.

This landing page was built to:

- Showcase the platform's vision and core features
- Provide a clean, fast, and responsive web presence
- Demonstrate how **data science students can leverage AI** to rapidly prototype and ship functional web products — no formal front-end training required
